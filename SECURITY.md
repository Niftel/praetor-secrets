# Security policy

Praetor Secrets Service is under active development and is not production-ready.

Do not report suspected vulnerabilities in public issues. Contact the Niftel
maintainers privately with the affected commit, reproduction steps, impact, and
any proposed mitigation. Do not include real credentials, private keys, database
dumps, or customer data.

Production readiness requires an independent review using
[`docs/security-review-packet.md`](docs/security-review-packet.md). Automated
security gates support that review but do not replace it.
